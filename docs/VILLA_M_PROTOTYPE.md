# Projet test — Villa M

## Informations projet

Nom : Villa M  
Type : Réaménagement de jardin ancien de villa  
Localisation : Casablanca  
Style souhaité : Méditerranéen contemporain  
Budget : non indiqué  
Contrainte principale : ne pas dénaturer l’existant complètement  

## Zones reconnues dans le prototype manuel

1. Jardin piscine / terrasse arrière
2. Grand jardin latéral
3. Façade principale / entrée
4. Passage portail / entrée secondaire
5. Niche latérale sous balcon / façade

## Concept global

Jardin méditerranéen contemporain conservateur.

L’objectif est de garder la structure existante : piscine, pelouses, haies, façades blanches, bois et certains sujets végétaux. L’intervention vise surtout à clarifier les massifs, moderniser les bordures, renforcer l’éclairage, améliorer les sols et unifier la palette végétale.

## Palette végétale possible

Arbres et sujets structurants : olivier, cyprès, chamaerops, pittosporum taillé, laurier noble, agrumes en pot, cycas existant si sain.

Arbustes : teucrium fruticans, westringia, myrtus communis, pittosporum tobira nana, raphiolepis, laurier-tin, romarin, santoline.

Vivaces et graminées : lavande, agapanthe, gaura, dietes, stipa, pennisetum, lomandra, liriope pour zones ombragées.

Sols : gravier beige clair, dalles beige clair, bordures acier corten ou pierre claire, pots anthracite ou terre cuite sobre.
