## Étape 2 — Créer un projet Supabase

Dans Supabase :
- New project
- Choisir un nom : landscape-ai-studio
- Garder le mot de passe en sécurité
- Attendre la création du projet

Ensuite :
- aller dans SQL Editor
- coller le contenu de `supabase/schema.sql`
- cliquer Run

## Étape 3 — Récupérer les clés Supabase

Dans Supabase :
- Project Settings
- API
- Copier : Project URL
- Copier : anon public key
- Copier : service_role key

## Étape 4 — Créer une clé OpenAI API

Dans OpenAI Platform :
- créer une API key
- la garder secrète

## Étape 5 — Mettre les clés dans `.env.local`

Créer un fichier `.env.local` à la racine du projet avec les clés.

## Étape 6 — Lancer le projet

Dans le terminal :

```bash
npm install
npm run dev
```
