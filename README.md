# Landscape AI Studio — Prototype V0.2

Prototype d'application IA pour architecte de paysage.

## Flux couvert

1. Upload photos
2. Analyse du site
3. SWOT
4. References / benchmark textuel
5. Idees par espace
6. Selection d'idee
7. Insertion dans le plan via plan masse + zones
8. Texturage 2D conceptuel
9. Prompt de plan realiste
10. Validation finale

## Limite honnete

Cette V0.2 ne produit pas encore automatiquement un vrai rendu image photorealiste du plan masse. Elle cree les donnees, le workflow et le prompt necessaire pour brancher une generation d'image en V0.3.
